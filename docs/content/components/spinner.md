Use `.spinner` with `role="status"` for loading indicators. Size with `.small` or `.large`.

{% demo() %}
```html
<div role="status" class="spinner small"></div>
<div role="status" class="spinner"></div>
<div role="status" class="spinner large"></div>
```
{% end %}
